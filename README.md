# backend_lecture

1. Устанавливаем uv
```pip install uv```

2. Создаём виртуальное окружение
```uv venv```

3. Активируем виртуальное окружение
```source .venv/bin/activate```

4. Устанавливаем зависимости
```uv sync```

5. Запускаем сервер!
```fastapi dev``` 

Чтобы увидеть документацию, перейдите по ссылке http://127.0.0.1:8000/docs#/
